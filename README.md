# 📊 Анализ продаж (SQL)

## 📌 Описание проекта
В данном проекте выполнен анализ данных о продажах с использованием SQL.  
Цель — изучить ключевые бизнес-показатели и найти инсайты.

---

## 📊 Данные
- Источник: датасет Superstore
- Количество строк: 9800
- Основные поля: заказы, клиенты, продажи, регионы, товары

---

## 🔍 Проведённый анализ

### 1. Базовые метрики
- Общее количество заказов
- Общая сумма продаж
- Средний чек

### 2. Лидеры
- Топ-3 клиента по сумме покупок
- Топ-3 города по продажам
- Самые прибыльные товары

### 3. Агрегации
- Продажи по регионам
- Продажи по категориям
- Средний чек по клиентам
